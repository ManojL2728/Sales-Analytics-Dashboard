# Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data using Python, MySQL, SQL, and Power BI.

The dashboard provides insights into:

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Revenue by State
- Revenue by Product Category
- Payment Type Distribution
- Revenue Trend Over Time

---

## Technologies Used

- Python
- Pandas
- MySQL
- SQL
- Power BI

---

## Dataset

Brazilian E-Commerce Public Dataset by Olist

---

## Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. Load Data into MySQL
4. SQL Analysis
5. Power BI Dashboard Creation
